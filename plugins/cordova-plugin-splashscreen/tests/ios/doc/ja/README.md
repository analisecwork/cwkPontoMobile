# CDVSplashScreen の iOS のテスト

`Node.js` `コルドバ`ios をプルするをインストールする必要があります。.

コルドバ ios をインストールします。

    npm install
    

現在のフォルダーに.

# Xcode からテスト

  1. `CDVSplashScreenTest.xcworkspace`ファイルを起動します。
  2. スキーム] ドロップダウン メニューから"CDVSplashScreenLibTests"を選択します。
  3. クリックし、`再生`ボタンを押し、テストを実行する`レンチ`のアイコンを選択

# コマンドラインからテスト

    npm test
